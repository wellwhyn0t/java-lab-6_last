# Лабораторная работа №6 - Вариант 18
## "Сортировщик по порядку" в Telegram

### Описание
Telegram-бот для викторины, где пользователю предлагается упорядочить элементы в правильном порядке.

**Вариант 18:** Сортировщик по порядку
- Бот предлагает упорядочить элементы (планеты по удалённости, события по хронологии и т.д.)
- Используются коллекции: `List` для последовательности, `HashMap` для проверки порядка, `Set` для отслеживания уникальных вопросов

### Структура проекта
```
src/main/java/ru/university/sortbot/
├── SortBotApplication.java    # Главный класс запуска
├── SortBot.java               # Основной класс бота
├── config/
│   └── BotConfig.java         # Конфигурация (токен, имя бота)
├── model/
│   ├── SortItem.java          # Модель элемента сортировки
│   ├── SortTask.java          # Модель задания
│   └── UserSession.java       # Сессия пользователя
├── manager/
│   └── SessionManager.java    # Менеджер сессий (HashMap)
└── game/
    └── QuizDataLoader.java    # Загрузка данных викторины
```

### Команды бота
- `/start` - Приветствие и описание игры
- `/quiz` - Начать новый раунд викторины
- `/stop` - Остановить текущую игру
- `/stats` - Показать статистику игрока
- `/help` - Подробная справка

### Примеры категорий заданий
1. **Планеты** - по удалённости от Солнца
2. **Материки** - по площади
3. **Римские цифры** - по возрастанию
4. **Времена года** - месяцы начала сезонов
5. **Цвета радуги** - в правильном порядке
6. **Ноты** - музыкальная гамма

### Как запустить

#### 1. Получите токен бота
1. Откройте Telegram и найдите @BotFather
2. Отправьте команду `/newbot`
3. Придумайте имя и username для бота
4. Скопируйте полученный токен

#### 2. Настройте конфигурацию
Откройте файл `src/main/java/ru/university/sortbot/config/BotConfig.java` и вставьте токен:

```java
public static final String BOT_TOKEN = "ваш_токен_от_botfather";
public static final String BOT_USERNAME = "username_вашего_бота";
```

#### 3. Запуск через Maven
```bash
./mvnw clean compile exec:java -Dexec.mainClass="ru.university.sortbot.SortBotApplication"
```

#### 4. Сборка JAR
```bash
./mvnw clean package
```
